ShijuCUIClient
================================================
5E実験のCUIクラアントです。
使用は自己責任でお願いします。


使い方
=================================================
コマンドラインで起動します。
起動時、画面の指示に従い、ユーザ名とサーバのアドレスを入力してください。
入力しない場合はデフォルトのユーザ名とアドレスを利用します。


既知のバグ
=================================================
バグだらけです。
